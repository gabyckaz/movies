# Movies

## Installation

1. Clone git project
2. Install pip3
3. Install python3

4. Install flask and flask-jwt-extended


```bash
pip3 install flask
pip3 install flask-jwt-extended
```

## Run
```bash 
python3 movies.py
```
It should run on http://127.0.0.1:5000/
